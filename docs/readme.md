AfriTalent

Présentation

AfriTalent est une plateforme web permettant de mettre en relation des freelances africains et des clients à la recherche de services professionnels.

Le projet a été réalisé dans le cadre du module de développement web en utilisant HTML, CSS, Bootstrap et JavaScript.

⸻

Fonctionnalités

* Navigation responsive avec Bootstrap
* Section Hero moderne
* Bento Grid pour la présentation des services
* Cartes de freelances avec effets hover
* Témoignages avec carousel Bootstrap
* Dark Mode avec sauvegarde dans localStorage
* Navbar dynamique au scroll
* Bouton “Retour en haut”
* Compteurs animés au scroll
* Animations Fade-In des sections
* Filtrage dynamique des freelances
* Validation du formulaire de contact
* Déploiement avec GitHub Pages

⸻

Technologies utilisées

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Git & GitHub
